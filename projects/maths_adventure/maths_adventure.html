Introduction:

This is a platformer with a maths theme for young children. The levels have a basic layout, and have a simple arithmetic question at the end. It is a project that I was commissioned to make.

Assets:

The game mostly just uses sprites and sounds from Super Mario Brothers. These were just what was provided by my client. The source code and all the used assets can be downloaded here. This includes the images, sounds, level data, and the font used.

About the Code:

This is a large project spanning many files and thousands of lines of code, so I'll not include it all here. You can view it in the download link if you are interested.

Most of the work is done in the update method of GameScene in scenes.py. This makes all the necessary method calls to other classes and essentially does all that is needed for the game to run. It also does things like check if the player has run out of lives, advance the level if the player has reached the end, and update the HUD to display the score, lives, and coins collected.

As an example, here is a snippet of code from that very class which kills the player. 

# Kill the player when they have no lives or fall out of the world
if self.player.lives <= 0 or self.player.rect.top > self.world_size[1]:
    # Stop all music and sounds
    pygame.mixer.music.stop()
    for sound in self.sounds.values():
        sound.stop()

    # Play death effect
    self.sounds['death'].play()
    self.dead = True  # Set dead flag
    # Causes the player to jump up in the air when dead
    self.player.velocity.y = -self.player.jump_force
