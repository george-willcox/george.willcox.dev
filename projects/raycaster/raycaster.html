Introduction:

For this program I am going to be using the following images, the first image of a wall will not be used in the untextured variant though.





These images can be downloaded here.

In addition to the images, a text file called 'level.txt' is used to define the level layout. It contains a series of 1's and spaces to show where the walls are, later we will add other numbers to this file to change the colour or the texture displayed on that wall.

The contents of the file are shown below, but it can also be downloaded here.
